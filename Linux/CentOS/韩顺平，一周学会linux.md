linux的特点：
```
    1.免费的，开源的
    2.支持多线程，多用户的
    3.安全性好
    4.对内存和文件管理优越
```

linux的文件系统是采用级层式目录结构：
```
    /       ：根目录
    /bin    ：系统命令目录
    /dev    ：设备目录
    /home   ：用户的主目录
    /lib    ：系统库目录      .so动态库      .a静态库      
    /media  ：多媒体目录
    /mnt    ：挂载目录
    /proc   ：内存目录
    /sbin   ：超级用户的命令目录
    /tmp    :临时文件目录
    /var    ：经常变化的文件目录
    /boot   ：启动分区
    /etc    ：配置文件目录  （不明白不要改动）  ，大多数软件和服务在这个目录下都有一个副本 
        在 /etc 文件中 ls -l 文件名  ：可以查看到文件在哪里映射过来的
    /lost+found     ：失物认领处
    /opt    ：第三方软件目录
    /root   ：root用户的家
    /usr    ：系统程序目录
```

vi编辑器：
```

    vi操作模式：
        1、Command Mode    命令模式：进入 vi 中首先是命令模式
        2、Insert Mode     输入模式：在 vi 中按 I 就进入了输入模式
        3、Last Line Mode  末行模式：按 ESC + ： 进入到末行模式

    开发步骤：
        1、vi 文件名  ：进入文件/创建文件在进入
        2、输入[进入插入模式]
        3、输入 esc [进入命令模式]
        4、输入 ：[wq 表示保存退出] 、[q! 表示强制退出不保存]、[q 表示退出]
```

命令：
```

    startx 启动图形化界面(我没用明白);
    shutdown -r now 立即重新启动计算机、reboot  立即重新启动计算机
    shutdown -h now 立即关闭计算机
    logout      注销用户
    pwd         当前位置
    useradd 用户名     添加用户
    passwd  用户名     设密码
    userdel 用户名     删除用户
    userdel -r 用户名     删除用户以及用户home中的文件
```

常用命令：
```

    ctrl + c    :结束
    ls      ：查看指定的文件夹下的所有文件
        ls /tmp/    查看 tmp 文件夹的文件
        ls -l   ：查看文件的详细信息
        ls -a   ：查看所有文件，包括隐藏文件
        查看之后：黑颜色的文件是普通文件、蓝颜色的是文件夹、红颜色的是源代码包

    pwd     ：查看当前工作目录
    cd      ：切换文件夹  写法 ：cd tmp
        cd ..   : 返回上一层
        cd ~    :返回用户自己的家
        
    clear       清屏
    login       登录
    logout      注销
    mkdir       创建目录
    rmdir       删除目录
    touch       创建一个空文件
    cp          复制命令
        cp -r dir1 dir2  递归复制命令（复制子目录信息）
    mv          移动文件和修改文件名
    rm          删除文件和目录
        rm -rf *  删除所有内容（保存目录和文件） r （递归）  f （强制）
    ln          创建符号连接
        ln  -s 源  目标
    more        显示文件内容，带分页
    less        显示文件内容，带分页
    grep        在文本中查询内容
    |           管道命令{ 在 linux 和 unix 系统中， | 就是管道命令：就是把上一个命令的结果，交个 | 的后面的命名去处理 }

    find        搜索文件及目录

    >           内容从定向，把查找的内容指定到某个文件中（覆盖）
    >>          内容从定向，把查找的内容指定到某个文件中（追加）
      grep -n "neirong" aa.php > a.txt { 查询aa.php中有没有“neirong” 有就保存到 a.txt中 }
